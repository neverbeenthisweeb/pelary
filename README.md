# pelary

Predict your salary and then win the negotiation!

### Requirements

- Python >=3.8.2

### Setting up

1. Install dependencies (`pip install -r requirements.txt`)
2. Set your environtment variables (`cp .env.example .env`)
3. Run _pelary_ (`python main.py`)
